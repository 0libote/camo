# Standard Camo Rules

This document outlines the standardized requirements for mastery camos across all weapon classes.

## Mastery Camos

These requirements apply universally to every weapon in the game.

| Camo Name | Requirement |
| :--- | :--- |
| **Military** | Kills (80x) |
| **Shattered Gold** | 3 Kills Without Dying (10x) |
| **Arclight** | Double Kills (10x) |
| **Tempest** | 5 Kills Without Dying (3x) |
| **Singularity** | Unlock Tempest (30x) |

## Special Camos

The requirements for **Diamondback**, **Raptor**, and **Mainframe** are weapon-specific or class-specific and are not covered by these global rules.
